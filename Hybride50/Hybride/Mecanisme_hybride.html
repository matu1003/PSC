Les deux mécanismes que vous avez découverts — Veto-Rank et Short Listing — ont chacun leurs forces et leurs faiblesses.

Pour tirer parti des avantages des deux, nous introduisons un mécanisme hybride qui combine les deux approches.

Le principe est simple : vous donnez toujours le même type d’input, quel que soit le mécanisme qui sera utilisé. Ensuite, le mécanisme appliqué est tiré au hasard entre VR et SL.

Input unique : vous classez les 5 options (animaux) selon votre stratégie.

Ce classement est ensuite interprété différemment selon le mécanisme tiré au sort :

  • Si le mécanisme est Veto-Rank :
    → Vos 2 derniers voeux sont considérés comme veto.
    → Les 3 premiers sont utilisés pour établir votre classement des restants.
  • Si le mécanisme est Short Listing :
    • Si vous êtes J1 : vos 3 premiers choix constituent la shortlist.
    • Si vous êtes J2 : on choisit, parmi les 3 proposés, celui que vous avez le mieux classé.
